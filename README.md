# Website-Status-Checker

### Instalasi
----
Membutuhkan [Node.js](https://nodejs.org/).
Install dan menjalankan aplikasi :

```sh
$ git clone https://github.com/BlackhatCode/Website-Status-Checker.git
$ cd Website-Status-Checker
$ npm install
$ node index
```
